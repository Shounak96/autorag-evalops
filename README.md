# AutoRAG EvalOps

AutoRAG EvalOps is an agentic RAG testing and monitoring platform that helps developers evaluate retrieval accuracy, citation coverage, answer grounding, latency, and prompt quality before deploying RAG applications.

## Tech Stack

- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: FastAPI, Python
- Database: PostgreSQL + pgvector
- CI/CD: GitHub Actions
- Deployment: Vercel + Render

## Core Features

- Document upload and ingestion
- Text chunking and embeddings
- RAG answer generation with citations
- Evaluation datasets and test cases
- Agentic verification workflow
- Dashboard for pass rate, retrieval score, citation coverage, and latency
- CI/CD quality gate for RAG evaluations
